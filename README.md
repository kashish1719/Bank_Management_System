# Banking Management System

A console-based Banking Management System developed in C++ using Object-Oriented Programming (OOPs) concepts and file handling. The project allows users to create bank accounts, securely log in, 
perform banking operations, and maintain transaction history with persistent data storage.

## Features:-
- Create new bank accounts
- Secure login authentication
- Deposit money
- Withdraw money
- Check account balance
- View account details
- Track transaction history
- Persistent data storage using file handling

## Technologies Used
- C++
- Object-Oriented Programming (OOPs)
- File Handling
- STL Vector
- Console-based UI
- OOP Concepts Implemented
- Classes & Objects
- Encapsulation
- Modular Programming
- Member Functions
- Data Abstraction

## Project Structure
├── main.cpp
├── KBOIxxxx.txt          # User account data files
├── Kashish-Bank-Of-India.txt   # Stores bank code/account count

## How It Works
1) User creates a bank account.
2) Account details are stored in text files.
3) User logs in using account number and password.
4) Banking operations update balance and transaction history.
5) Updated data is saved automatically using file handling.
